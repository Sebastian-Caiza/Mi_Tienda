# MiTienda - Sistema de Ventas

## Descripción
MiTienda es un sistema de ventas de escritorio desarrollado en JavaFX que permite gestionar productos mediante un panel administrativo con 
autenticación de usuarios por rol.

## Tecnologías utilizadas
- Java 21
- JavaFX 21
- Scene Builder
- Maven
- IntelliJ IDEA

## Funcionalidades implementadas
- Login con validación de usuario, contraseña y rol
- Dashboard con menú lateral de navegación
- Módulo de productos con formulario completo
- CRUD de productos (Crear, Leer, Actualizar, Eliminar)
- Tabla de productos con columnas: Código, Nombre, Categoría, Precio, Stock, Estado
- Selección de producto desde la tabla para editar
- Validación de campos obligatorios
- Estilos CSS personalizados

## Instrucciones para ejecutar
1. Clonar el repositorio
2. Abrir el proyecto en IntelliJ IDEA
3. Verificar que el JDK 21 esté configurado
4. Ejecutar MainApp.java
5. Ingresar con las credenciales:
   - Usuario: admin
   - Contraseña: 1234
   - Rol: Administrador
